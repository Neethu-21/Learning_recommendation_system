## 🔗 Live Demo
https://neethu-21-learning-recommendation-system-appapp-setott.streamlit.app/

## 📌 Features
- Student performance analysis
- Subject-wise trend visualization
- Risk prediction
- Personalized learning recommendations
- Study resources (PDF + YouTube)
- Student comparison dashboard

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## 🚀 How it works
The system analyzes student scores, identifies weak subjects, predicts performance, and recommends personalized learning paths.
